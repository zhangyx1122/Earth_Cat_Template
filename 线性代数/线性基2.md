# 线性基

线性基 能表示的线性空间与原向量 能表示的线性空间等价



用高斯消元得到线性基

先输入数组a[] 中

```cpp
int n, k;
ll a[N];

void getVec() {
    k = 0;

    for(int i = 62; i >= 0; -- i) {
        for(int j = k; j < n; ++ j) {
            if(a[j] >> i & 1) {
                swap(a[j], a[k]);
                break;
            }
        }
        if(!(a[k] >> i & 1)) continue;
        for(int j = 0; j < n; ++j) {
            if(j != k && (a[j] >> i & 1)) {
                a[j] ^= a[k];
            }
        }
        ++k;
        if(k == n) break;
    }

}

```

这里注意最后的线性基是a[]中从0到k-1个，在前的是**高位**
